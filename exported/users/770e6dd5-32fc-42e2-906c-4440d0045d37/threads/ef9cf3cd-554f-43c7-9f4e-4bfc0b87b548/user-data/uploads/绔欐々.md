站桩是武术和养生中的基本功，核心在于“中正安舒”与“内外合一”。虽然门派（如太极、意拳、八卦掌）的细节略有不同，但大体的要点可以归纳为以下几个核心方面：

### 1. 身体姿态（形）

这是站桩的基础，要求身体像一棵树一样，既有向下的扎根感，又有向上的生长感。

* **头顶悬（虚领顶劲）：** 想象头顶中心有一根线轻轻向上提拉，下巴微收，使颈椎自然拉直。
* **松肩坠肘：** 肩膀完全放松，不要耸起；肘部微下垂，感觉双臂下沉但有圆撑之力。
* **含胸拔背：** 胸部不挺、不凹，保持自然舒展；后背脊柱微向后张，使气机顺畅。
* **屈膝松胯：** 膝盖微屈（不超过脚尖），胯部向下放松，像半坐在高凳上，脊椎尾端下垂（垂闾）。
* **足底稳固：** 双脚平行或呈外八字（视功法而定），重心落在涌泉穴（脚心偏前）或脚掌中心，感觉脚趾轻轻抓地。

---

### 2. 呼吸与内感（气）

站桩不是静止的雕塑，而是“静中求动”。

* **腹式呼吸：** 采用自然顺腹式或逆腹式呼吸。要求：**深、长、细、匀**。舌尖轻抵上腭。
* **气沉丹田：** 呼吸时感觉气息引向小腹（脐下三寸处），不要憋气。
* **全身放松（松而不懈）：** 肌肉要放松，但骨架要撑起。这就是常说的“骨架挺拔，肌肉下垂”。

---

### 3. 意念引导（神）

站桩最忌讳心猿意马，需要通过意念来调控身心。

* **内视自省：** 闭眼或垂帘（留一线光），观察体内的紧张点并逐一放松。
* **意象引导：** * **抱圆：** 想象双臂环抱一棵大树或一个大气球，既怕抱碎了，又怕它掉下去。
* **入地：** 想象双脚像树根一样扎入地下数米。


* **平心静气：** 排除杂念，保持神志清醒但心境平和。

---

### 4. 站桩的阶段性禁忌

为了避免受伤或收效甚微，请注意：

| 阶段     | 重点                 | 注意事项                                 |
| -------- | -------------------- | ---------------------------------------- |
| **初期** | 找准姿势，增加时长   | 避免双腿发抖时过度硬撑，防止膝盖受损。   |
| **中期** | 寻找“整劲”，体会呼吸 | 避免过分追求意念，导致头晕或“火气上攻”。 |
| **长期** | 养成习惯，身心合一   | 站桩后不要立即洗冷水澡或吃生冷食物。     |

### 站桩后的收功

站桩结束时，不要立刻走动。建议先双手叠扣在丹田处静养片刻，然后轻轻拍打全身肌肉，最后慢走几步，让身体逐渐恢复常态。
